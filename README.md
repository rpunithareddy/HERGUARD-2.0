HerGuard is a women safety web application that enables users to send **instant SOS alerts with live location tracking** for quick emergency response.

## Features
- One-click SOS alert  
- Real-time location access  
- Google Maps redirection  
- Secure Flask backend  
- SOS data logging  

## Tech Stack
HTML, CSS, JavaScript, Flask, SQLite

## Purpose
To provide a **fast, simple, and accessible safety solution** for women during emergencies.

> Safety at the press of a button.
